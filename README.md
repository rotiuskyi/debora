# Debora

Elixir/Phoenix (backend) and React (frontend) application.

## React

React app stuff placed to `webui` directory. To get started developing you have to install `docker` and `compose` plugin.

Then just run:
```
docker compose up -d
```
For older versions:
```
docker-compose up -d
```
